# 🍽️ Flavor Hub — Restaurant Landing Page

A modern, responsive landing page for a fictional restaurant, built with HTML, CSS & Tailwind CSS.

## 🖼️ Preview

![Screenshot](docs/screenshot.png)

## 🚀 Features

- Fully responsive design (mobile-first)
- Hero section with CTA
- Menu section with card-based layout
- Customer testimonials section
- Reservation form
- Smooth scroll navigation
- Fixed navbar with hamburger menu on mobile
- Clean, modern UI with Tailwind CSS

## 🛠️ Tech Stack

- HTML5
- CSS3
- Tailwind CSS (CDN)
- Google Fonts (Cairo)

## ⚙️ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/mahmoudmagdyedu/Restaurant-Landing-Page.git
   ```
2. Open `index.html` in your browser.

No build tools required — pure HTML & CSS!
